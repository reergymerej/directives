Isolated Scope

controller scope vars

  • $scope.a = 'aaa';
  • $scope.b = 'bbb';
  • $scope.c = 'ccc';

directive reference in view

<ex-isolated a="b" x="c" y="z" ></ex-isolated>

The view has access to its variables.

  • a: {{ a }}
  • b: {{ b }}
  • c: {{ c }}

The directive maps its scope variables with

scope: {
// make a available in the directive's scope and map it to the controller's variable specified by the attribute a
a: '=',
// make foo available in the directive's scope // and map it to the controller's variable specified // by the attribute x foo: '=x' }
